# Docker Desktop Feedback

[![GitHub pinned issue](https://img.shields.io/badge/github-find%20an%20issue-blue.svg?logo=github&style=flat-square)](https://github.com/docker/desktop-feedback/issues/4)

Welcome to the unified issue tracker for **Docker Desktop** on **Windows**,
**macOS**, and **Linux**. This repository is where you can report bugs and share
feedback about your experience with Docker Desktop across all supported
platforms.

## 🔽 Download Docker Desktop

Docker Desktop is free to download:

👉 [Get Docker Desktop](https://www.docker.com/products/docker-desktop/)

## 📚 Documentation

Looking for how-to guides or troubleshooting tips?

Check out the official
[Docker Desktop documentation](https://docs.docker.com/desktop/) - it's the best
place to get detailed answers.

## 💬 Support

- **Paid support** is available to Docker **Pro** and **Team** subscribers via
  the [Desktop support form](https://hub.docker.com/support/contact/). (Login
  required)
- This repository is intended for **bug reports** and **product feedback only**.
- **Support requests** (e.g. installation problems, usage questions) will be
  closed or converted to Discussions.
- For help from the community, visit the
  [Docker Community Slack](https://dockercommunity.slack.com/) (Not a member
  yet?
  [Join Here](https://communityinviter.com/apps/dockercommunity/docker-community))

## 🐛 Reporting Issues

If you run into a bug or problem:

1. Use the **Search bar** (press `s`) to check for existing reports.
2. If your issue hasn’t been reported, click **New issue** and fill out the
   provided template.

This helps us prioritize and fix problems more efficiently.

## 🧩 Open Source Components

Docker Desktop includes many open source components. You can find a complete
list in the app:

> **Docker Desktop → 🐳 Menu → About Docker Desktop → Acknowledgements**

Note: Component usage may vary by platform.
